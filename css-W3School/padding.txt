Exercise 1:
Set the top padding of the <h1> element to 30 pixels.
<style>
h1 {
  padding-top: 30px;
}
</style>
<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph</p>
  <p>This is a paragraph</p>
</body>

Exercise 2:
Set the padding of all sides of the <h1> element to 30 pixels.
<style>
h1 {
  padding: 30px;
}
</style>

<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph</p>

</body>

  padding: 40px 10px;
}
</style>
